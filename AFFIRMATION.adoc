=== The new layer: constrained-form metatheory (`src/abi/Lang/*`)

Machine-checked, over a faithful model of `lib/ast.ml` / `lib/eval.ml` /
`lib/constrained_check.ml` (landed in PRs #73–#77; mapped in
link:src/abi/Lang/README.adoc[`src/abi/Lang/README.adoc`]):

[cols="1,3",options="header"]
|===
| Guarantee | Headline (proved)
| Termination | the evaluator is a TOTAL function — loops via a `Nat` count from static bounds; calls via the rank-stratified `evalLevel` over the acyclic call graph (`Lang.Eval`)
| Checker soundness | a `Ranking` rules out direct & mutual recursion (`Lang.Checker`)
| Reversibility | `inverse` is an involution on `incr`/`decr`/`swap`; `assign` is the irreversible `echo[A,B]` boundary (`Lang.Reversible`)
| Resource + accountability | the static budget IS the accountability-trace length, both total (`Lang.Resource`)
|===

=== The honest nuance you must not lose

* *These proofs are of a MODEL, not the OCaml.* The Idris2 `Lang.*` syntax /
  semantics / checker faithfully *model* `lib/ast.ml` etc., but the model-fidelity
  gap is real.  Closing it — 3b checker completeness, 4b general reversibility,
  5b runtime-trace faithfulness, and ultimately verified extraction — is tracked
  in *https://github.com/hyperpolymath/oblibeny/issues/78[#78]*.  Reading
  "constrained form proven sound" as "the OCaml is proven sound" is over-reading.
* *Crypto is an external trust boundary.* The post-quantum signature
  completeness/soundness live in linked C (`liboqs`/`libsodium`); `liboqs` is
  absent here, so the FFI does not link.  `obli-pkg verify` derives the canonical
  payload and is fail-closed (PR #72) but has no matching *signer* yet.

== Reproduce it yourself

From the repo root, at the commit above (`just` not required):

[source,sh]
----
export PATH=/opt/idris2-install/bin:$PATH      # idris2 is installed off PATH
idris2 --build src/abi/oblibeny-abi.ipkg        # ABI + Lang.* proofs; expect exit 0
dune build && dune test                          # 27-test conformance; expect exit 0
dune exec -- oblibeny examples/hello.obl --dump-trace   # accountability trace
grep -rnE 'believe_me|assert_total|idris_crash' src/abi/Crypto.idr src/abi/Packages src/abi/Lang  # expect: nothing
for f in ffi/zig/src/*.zig; do zig ast-check "$f"; done  # FFI compile-check (not a link)
----
