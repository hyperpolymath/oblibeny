Linear Types for Safety


(def transfer-funds (from: Account, to: Account, amount: Money) 
                    -> (IO (Result Unit))
  ; Money is a linear type - can't be duplicated or lost
  (with-transaction
    (let funds (withdraw from amount))  ; from loses ownership
    (deposit to funds)                  ; funds transferred, can't be reused
    (Ok ())))

== What is Oblivious Computing?

Oblivious computing hides access patterns and data access from observers,
providing strong privacy guarantees beyond encryption alone. These tools
enable building privacy-preserving applications.
