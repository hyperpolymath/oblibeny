== Bottom line — where we are

Oblíbený sits at *~85%* (`STATE.a2ml`, recalibrated *down* from a prior inflated
100% once PR #51's CI gate exposed non-compiling proof stubs — an honest figure).
The *language core is solid and live-green this session*: build, the 27-test
conformance suite, and an actually-traced golden-path run. The whole live
frontier is in the *INTEND* tier and is mostly *not* agent-actionable: the
post-quantum crypto FFI compiles but cannot link in CI (external `liboqs`),
`obli-pkg verify` is a deliberate MVP stub, and three housekeeping intents are
maintainer-only. The *WISH* tier — the `obli-*` sibling ecosystem and deeper
formalisation — is unstarted horizon work.

The single honesty gap in *this* snapshot vs a complete one: the Idris2 ABI
proof MUST was *not run here* (no `idris2`), only carried forward as CI-gated.
